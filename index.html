Favorite Programming Language

I'm a big fan of Julia () because it makes it easy to write one-off scripts and experiments, while having lots of capabilities for writing reasonably high-performant code.

Julia is a dynamically typed, JIT compiled language, making it easy to use when being lazy, but also very efficient when writing highly performant code.

Particularly, a the popular CUDA.jl library serves as a low barrier to entry introduction for GPU programming with NVIDIA GPU's. To give an example, this is how you copy an array from "CPU memory" (RAM) to "GPU memory" (DRAM) in CUDA C++:

int h[5] = {5, 4, 3, 2, 1}; // host pointer
int size = 20; // 5 numbers * 4 bytes

int *d; // device pointer
cudaMalloc(&d, size);
cudaMemcpy(d, h, size, cudaMemcpyHostToDevice);

// do stuff with d

cudaFree(d); // free device memory when done

And this is the equivalent code in Julia using CUDA.jl:

h = [5, 4, 3, 2, 1]
d = cu(h)

# do stuff with d

By just calling cu, CUDA.jl handles all of the memory allocation and copying, and Julia's garbage collector handles freeing unused objects. Additionally, debugging is (most of the time) a lot easier in Julia than C++.
